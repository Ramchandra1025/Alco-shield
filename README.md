# 🚀 AlcoShield  
### Smart Alcohol Control System using AI + Full Stack

Excited to share a project my team and I worked on — **AlcoShield**, a system designed to monitor and regulate alcohol consumption using real-time signals.

---

## 💡 The Idea
Move beyond manual tracking and build an intelligent system that detects alcohol-related activity across multiple sources and helps users stay within safe limits.

---

## 🔍 Key Features

- 🧾 **AI-powered OCR** to scan bills and detect alcohol items  
- 📱 **SMS monitoring** for transaction insights  
- 📍 **GPS-based proximity alerts** near alcohol vendors  
- 💳 **Smart payment gate** to restrict purchases beyond set limits  
- 📊 **Real-time consumption tracking** with risk levels  
- 🤖 **Behavioral intervention alerts** (context-aware nudges)

---

## 🧠 Tech Stack

- **Backend:** Python (Flask)  
- **AI / OCR:** Claude Vision, EasyOCR, fallback detection layers  
- **Database:** Supabase  
- **Frontend:** HTML, CSS, JavaScript  
- **Additional Systems:** Geolocation services + rule-based detection engine
